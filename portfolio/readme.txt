   1. What Each HTML Element Does 

  MY webpage contains various HTML elements that serve specific functions:-

• <!DOCTYPE html>: Defines the document type and version. 
• <html>: The root element that encapsulates all webpage content. 
• <head>: Contains metadata, such as the title, character set, viewport settings, and linked 
    resources (CSS, icons). 
• <meta charset="UTF-8">: Specifies character encoding for proper text display. 
• <meta name="viewport" content="width=device-width, initial-scale=1.0">: Ensures 
   responsiveness on different screen sizes. 
• <title>: Sets the webpage title displayed in the browser tab. 
• <link rel="icon">: Links an icon for the browser tab. 
• <body>: Contains the visible webpage elements. 
• <nav>: Defines the navigation menu for moving between pages. 
• <table>: Used to structure navigation links and display tabular data for work experience 
    and projects. 
• <tr> / <td>: Define table rows and cells respectively. 
• <ul> / <ol>: List elements for skills and education. 
• <li>: Represents an item in a list. 
• <img>: Displays profile images. 
• <h1>, <h2>, <h3>, <h4>, <h5>, <h6>: Headings used to organize content hierarchically. 
• <p>: Paragraph element for textual content. 
• <hr>: Horizontal rule for section separation. 
• <form>: Defines an input form for user interaction. 
• <input>: Creates various input fields in the contact form. 
• <textarea>: Allows users to enter multiline text. 
• <button>: Submits the form. 

2. Why You Chose Specific Input Types 

    MY form contains different input types for better user experience:- 

• <input type="text">: Captures a user's name. 
• <input type="email">: Ensures valid email format. 
• <input type="tel" pattern="[0-9]{10}">: Accepts only a 10-digit phone number. 
• <textarea>: Provides space for longer messages. 
• <input type="checkbox">: Lets users select multiple preferred contact methods. 
• <input type="radio">: Restricts selection to one inquiry type. 
• <input type="date">: Allows users to specify a preferred contact date. 
• <input type="range">: Gives users a sliding scale for urgency, improving usability. 
• <button>: Submits the form.


3. How Your Navigation Structure Works

    MY navigation system is implemented in two ways:- 
    
1. Table-Based Navigation 
  o <table> organizes navigation links (<a> elements) in a structured layout. 
  o Clicking a link redirects users to different pages. 
2. Dropdown Navigation (Contact Page) 
  o <select> creates a dropdown menu with page options. 
  o The onchange event automatically redirects users upon selection.
